WSL Setup

Set mount umask

# /etc/wsl.conf
[automount]
enabled = true
options = "metadata,umask=022"

Documentation for the /etc/wsl.conf is *here

Set ssh-agent

1st: Open powershell as administrator, and run:

PS C:\> Start-Service ssh-agent
PS C:\> Get-Service ssh-agent

Status   Name               DisplayName
------   ----               -----------
Running  ssh-agent          OpenSSH Authentication Agent

2nd: Add your keys using the ssh-add command at the powershell

3rd: Download the latest ssh-agent-wsl version (to a windows folder) here

4th: Uncompress it to a folder of your choice (on windows side): C:\Users\loop0\bin

5th: Add the following to your .bashrc in the WSL user’s home

eval $(/mnt/c/Users/loop0/bin/ssh-agent-wsl -r)
